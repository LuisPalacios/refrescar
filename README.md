# Refrescar

Un pequeño repositorio para refrescar mi C/C++ en el 2024, con cmake, boost, google tests

### 1-hola

El típico hola mundo preparado para cmake. 

```shell
cd 1-hola
cd build
cmake ../src
make
./hola
```

### 2-cuadrado tests

Un ejemplo de uso de Unit Tests con Google Tests

```shell
cd 2-cuadrado
cd build
cmake ../src
make
./cuadrado
```
